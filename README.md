# Zaxaroplasteio
Πως να τρέξεις το site:
1. θα χρειαστεί να προσθέσουμε στον φάκελο μας xampp -> htdocs -> project όλον τον φάκελο project που υπάρχει ανεβασμένος στο github.
2. Στην συνέχεια θα πρέπει να ανοίξουμε στο notepad++  όλα τα αρχεία doctype.html,epikinonia.html,proionta.html, το style.css, το main.js καθώς επίσης κι όλα τα php, που περιλαμβάνουν τα action_page.php,cart.php,checkout.php,library.php,order.php,page.php,page2.php,pageregister.php,product.php
3. Θα χρειαστεί να ανοίξουμε το xampp control κι εκει να ενεργοποιησουμε την επιλογή apache, καθώς επίσης κι την επιλογή mysql.
4. Θα ανοίξουμε στον browser μας την διεύθυνση του localhost server μας η οποία είναι: http://localhost/project/doctype.html
5. Επίσης θα πρέπει από το xampp να πατήσουμε στο mysql την επιλογή admin ώστε να μας πάει στο localhost phpmyadmin στο οποίο θα χρειαστεί να κάνουμε εισαγωγή την βάση μας η οποία βρίσκεται στον φάκελο project με το όνομα zaxaroplasteio.sql ώστε να παίρνει τα δεδομένα των χρηστών κι να τα κρατάει.
6. Έχοντας λοιπόν όλες αυτές τις ρυθμίσεις μπορούμε να τρέξουμε το site μας.
